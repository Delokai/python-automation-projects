# Day 18 — JSON User Manager

This project introduces JSON storage by building a simple user management system.

## What I Learned
- What JSON is and why it’s used
- How to save data to a JSON file
- How to load JSON data back into Python
- How dictionaries map to JSON structure
- Building a small data-driven application

## Features
- Add users with a username and age
- Store users permanently in `users.json`
- Reload all users automatically on startup
- Display all stored users with formatting

## How to Run
python user_manager.py
